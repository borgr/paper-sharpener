# writing_skills
This project releases agentic skills to improve academic writing. It is based on the [evergrowing writing tips document](https://docs.google.com/document/d/14Wax8M5w8F_8miDlYJ9-I6wqpelxlXjCEUbkNzNMqqE/edit?usp=sharing) plus improvements specific for the skills (e.g. reviewers)

# use
If you are using claude code, for example, clone it under your .claude dir

# contribute
This is an evergrowing skill, contributions welcome
